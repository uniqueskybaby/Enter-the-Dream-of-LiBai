# 入梦李白

沉浸式全景诗词探索游戏。以李白经典诗篇为蓝本，通过 360° 全景场景、分支叙事、谜题解谜和 AI 诗词生成，让玩家"走进"诗境。

## 内置梦境

| 梦境 | 诗篇 | 诗句 |
|------|------|------|
| 庐山银河梦境 | 望庐山瀑布 | 飞流直下三千尺，疑是银河落九天 |
| 月影三人梦境 | 月下独酌 | 举杯邀明月，对影成三人 |
| 黄河天海梦境 | 将进酒 | 黄河之水天上来，奔流到海不复回 |
| 长江天际梦境 | 黄鹤楼送孟浩然之广陵 | 孤帆远影碧空尽，惟见长江天际流 |

此外，玩家可通过 AI 面板生成自定义诗境并永久保存。

## 核心功能

- **全景探索** — 基于 Pannellum 的 360° 场景漫游，含热点交互
- **分支叙事** — 每个场景有多条故事分支，导向不同结局
- **诗意谜题** — 收集碎片解锁星河诗阵
- **成就系统** — 首次入梦、速通、全结局收集等成就
- **资源奖励** — 月光、诗卷、宝珠等虚拟资源
- **诗词对决** — 结局页面的互动诗词玩法
- **AI 诗境生成** — 接入 OpenAI 兼容接口，输入任意诗词即可生成对应梦境
- **音频系统** — 背景音乐与场景音效

## 技术栈

- React 18 + TypeScript
- Vite 6
- Pannellum (全景引擎)
- Lucide React (图标)

## 快速开始

### plan A

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建产物
npm run preview
```

### plan B

####下载项目到本地后，点击start.bat。


## AI 诗境生成

在游戏内打开 AI 面板，输入诗词内容，系统会调用配置的 AI 接口生成全景梦境配置。生成的梦境会保存到 `public/data/` 目录，可永久游玩。

需要在服务端配置 OpenAI 兼容的 API 地址和密钥。
